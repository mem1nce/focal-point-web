# Focal Point 🍅
Focal Point, odaklanmanızı artırmak ve çalışma süreçlerinizi profesyonelce yönetmeniz için tasarlanmış minimalist bir Pomodoro uygulamasıdır.

## 🚀 Özellikler
- **Modüler Zamanlama:** Odak, Kısa Mola ve Uzun Mola modları.
- **Görev Yönetimi:** Görevlerinizi ekleyin, takibini yapın ve tamamlayın.
- **İstatistik Takibi:** Günlük çalışma süreleri ve seri (streak) takibi.
- **Modern Arayüz:** Karanlık mod destekli, temiz ve şık tasarım.

## 🛠 Kullanılan Teknolojiler
- HTML5 & CSS3 (Modern Flex/Grid düzeni)
- Vanilla JavaScript (Local Storage desteği)

## 💻 Geliştirme Notları
Bu proje, yazılım geliştirme disiplinini ve proje mimarisini öğrenmek amacıyla geliştirilmiştir.
